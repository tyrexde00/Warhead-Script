# Warhead-Script

Warhead Sequence Button Access Control for Armed Containment Area 13 V3

The script was originaly made by Marc but strongly edited by me!

I made the Alpha and Omega Warhead Sequence Buttons restricted to authorized personnel only using a script I wrote myself. No more random people triggering a site-wide nuke.

Currently, it only works with GuthSCP Keycards and DarkRP job teams.

You'll need GuthSCP Base, the GuthSCP Keycards, and of course the appropriate map Armed Containment Area 13 V3!
It only works in DarkRP.

Check Modes
The script supports three modes configurable at the top of the file:
both — Player needs the correct job AND a valid keycard in hand
keycard — Only the keycard is checked, job doesn't matter
job — Only the job is checked, keycard doesn't matter
